DESCRIPTION:
The administration is via an infusion (requires INFUSION RATE or INFUSION DURATION column-type in the data set).
The PK model has one compartment (volume V) and a Michaelis-Menten elimination (Vm, Km).


[LONGITUDINAL]
input = {V_logit, Vm_logit, Km_logit}

EQUATION:
odeType = stiff
; Limites para el parametro V_logit
a = 1e-3
b = 100
; Limites para el parametro Vm
c = 1e-3
d = 100
; Limites para el parametro Km
e = 1e-3
f = 100
; Modelo
V_bound = a + V_logit*(b-a)
Vm_bound = c + Vm_logit*(d-c)
Km_bound = c + Km_logit*(d-c)
; PK model definition
Cc = pkmodel(V = V_bound, Vm = Vm_bound, Km = Km_bound)
Y = log(Cc)

OUTPUT:
output = Y
